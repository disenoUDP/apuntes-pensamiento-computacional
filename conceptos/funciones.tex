\chapter{Funciones}

Las funciones también son conocidas en inglés como functions.

\section{Definición}

Las funciones permiten agrupar bloques de código para poder utilizarlos y volver a utilizarlos, sin necesidad de escribir el mismo código una y otra vez.

\section{Sintaxis}

Las funciones en JavaScript se declaran usando la palabra reservada $function$.

\begin{Code}
    function sumar(a, b) {
        let x = a + b;
        return x;
    }
\end{Code}
